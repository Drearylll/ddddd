1. **`get_real_location()`** - 主函数

   - 随机选择一个
   - 获取详细信息

2. **`_get_poi_details()`** - 获取地点详情
   - 获取图片 URL
   - 生成智能描述

3. **`_fetch_poi_image()`** - 从高德 API 获取图片
   - 调用地点详情接口
   - 提取 photos 中的图片

4. **`_generate_description()`** - 生成智能描述
   - 根据类型生成个性化文案
   - 添加距离和步行时间
   - 简化地址信息

5. **`_simplify_address()`** - 地址简化
   - 提取区名（黄浦区、徐汇区等）
   - 提取路名（南京东路、福州路等）
