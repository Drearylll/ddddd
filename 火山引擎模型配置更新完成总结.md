## 🎯 解决方案

### 方案 1：在火山引擎控制台开通服务（推荐）

**步骤**：

1. **登录控制台**
   ```
   https://console.volcengine.com/ark
   ```

2. **进入"我的模型"**
   - 查看可用的模型列表
   - 确认是否有 Doubao-Seed、Seedance、Seedream 系列

3. **开通服务**
   - 找到需要的模型
   - 点击"开通服务"
   - 同意服务协议

4. **检查 API Key**
   - 进入"接入点管理"
   - 确认 API Key 有效
   - 复制完整的 Key

5. **验证余额**
   - 进入"费用中心"
   - 查看账户余额
   - 确保有足够额度

### 方案 2：使用阿里云作为主服务

如果火山引擎配置复杂，可以暂时使用阿里云：

**修改** `services/doubao_ai.py`：

```python
def __init__(self, api_key=None, use_backup=True):  # 改为 True
    """默认使用阿里云备份"""
    self.config = get_backup_config()
    self.use_backup = True
```

这样会自动使用阿里云百炼，不需要额外配置。

### 方案 3：继续使用模拟数据

如果只是想测试功能，可以依赖降级机制：

```python
# 火山引擎失败后会自动降级到模拟数据
result = run_ai_timeline(user_id="xxx")
# 会返回模拟的文案和图片
```
