```
   ↓

   ↓

   ↓
7. 点击"提交创作"
   ↓
8. 保存到 localStorage
   ↓

```

## 🎯 使用方式

### 步骤 1: 进入共创

**在看看页面**:
1. 找到任意内容卡片
2. 点击"共创"按钮
3. 选择模式：
   - 输入 `1` - 接力模式
   - 输入 `2` - 夺擂模式

### 步骤 2: 创作内容

**接力模式**:
- 阅读原内容
- 在编辑器中续写下一段/下一格
- 点击"提交创作"

**夺擂模式**:
- 等待 AI 生成新要求（约 2 秒）
- 阅读新的创作要求
- 在编辑器中重新创作
- 点击"提交创作"

### 步骤 3: 查看结果

**提交后**:
- ✅ 显示"创作成功"提示
- ✅ 自动返回主界面
- ✅ 创作结果保存到 localStorage

**查看创作记录**:
```javascript
// 在浏览器控制台查看
const keys = Object.keys(localStorage).filter(k => k.startsWith('co_create_result_'));
keys.forEach(key => {
    console.log(JSON.parse(localStorage.getItem(key)));
});
```

## 🔧 技术实现

### 路由配置
```python
@app.route('/co_create')
def co_create_page():
    """共创编辑页面"""
    return render_template('co_create.html')
```

### JavaScript 逻辑

**初始化**:
```javascript
window.addEventListener('DOMContentLoaded', () => {
    // 从 localStorage 获取信息
    contentId = localStorage.getItem('co_create_content_id')
    mode = localStorage.getItem('co_create_mode')
    originalTitle = localStorage.getItem('co_create_title')
    originalContent = localStorage.getItem('co_create_content')
    
    // 更新界面
    document.getElementById('modeBadge').textContent = 
        mode === 'relay' ? '🔄 接力模式' : '🏆 夺擂模式'
})
```

**夺擂模式 AI 生成**:
```javascript
async function generateChallengePrompt() {
    const response = await fetch('/api/generate_more_content', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
            type: contentType,
            location: '附近',
            mood: '日常'
        })
    })
    
    const data = await response.json()
    // 更新原内容和创作要求
}
```

**提交创作**:
```javascript
function submitCoCreate() {
    const userContent = editor.value.trim()
    
    // 保存结果
    const result = {
        contentId,
        mode,
        originalTitle,
        originalContent,
        userContent,
        timestamp: new Date().toISOString()
    }
    
    localStorage.setItem(coCreateKey, JSON.stringify(result))
    alert('✅ 创作成功！')
    window.location.href = '/app_main'
}
```
