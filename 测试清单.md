## 🔧 常见问题排查

### 问题 1：看到欢迎页

**原因：** 代码未正确修改

**解决：**
```python
# 确保第 988-998 行代码正确
if not user_data.get('preferences'):
    # 自动生成默认偏好
    default_prefs = {...}
    save_user_data('preferences', default_prefs)
```

---

### 问题 2：只生成 2 条内容

**原因：** 代码未更新

**解决：**
```python
# 确保第 1075-1077 行代码正确
num_posts_to_generate = random.randint(3, 5)
print(f"[LIFE] First time generation: {num_posts_to_generate} posts (initial experience)")
```

---

### 问题 3：所有用户看到相同内容

**原因：** session 配置问题

**解决：**
```python
# 确保设置了 secret_key
app.secret_key = 'goin_immersive_mvp_2026'
```

---

### 问题 4：地点仍然是"浦东"

**原因：** 使用了旧版内容池

**解决：**
检查 `SCREENSHOT_CONTENT` 和 `ABNORMAL_CONTENT` 是否使用 `random.choice(PUDONG_LOCATIONS)`
