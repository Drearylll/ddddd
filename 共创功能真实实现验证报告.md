## 🔍 发现的根本问题

### 问题 1：函数重复定义
**位置**: `templates/app_main.html` 870 行 和 1118 行  
**问题描述**: 
- 第一个 `openCoCreate` 函数（870 行）使用 `prompt()` 弹窗选择模式
- 第二个 `openCoCreate` 函数（1118 行）使用模态框选择模式
- 两个函数同时存在，导致行为不一致

**解决方案**: 
- 注释掉 870 行的 prompt 版本
- 保留并使用 1118 行的模态框版本
- 在 `confirmCoCreate()` 函数中添加跳转逻辑

```python
@app.route('/api/co_create_generate', methods=['POST'])
def api_co_create_generate():
    """共创内容生成 API"""
    data = request.get_json() or {}
    prompt = data.get('prompt', '')  # 用户输入的要求
    content_type = data.get('type', 'novel')  # novel 或 comic
    
    ai_service = get_ai_service()
    
    # 根据用户要求生成内容
    if content_type == 'novel':
        result = ai_service.generate_novel(
            location='附近',
            mood='日常',
            user_profile=prompt  # 使用用户输入作为提示
        )
    elif content_type == 'comic':
        result = ai_service.generate_comic_script(
            location='附近',
            theme=prompt  # 使用用户输入作为主题
        )
    
    return jsonify({
        'success': True,
        'type': content_type,
        'data': result
    })
```

**路由 3: 发布内容 API**
```python
@app.route('/api/publish_co_create', methods=['POST'])
def api_publish_co_create():
    """发布共创内容到看看页面"""
    data = request.get_json() or {}
    
    co_create_data = {
        'content_type': data.get('content_type', 'novel'),
        'title': data.get('title', '共创作品'),
        'content': data.get('content', ''),
        'mode': data.get('mode', 'relay'),
        'original_content': data.get('original_content', ''),
        'user_contribution': data.get('user_contribution', ''),
        'created_at': datetime.now().isoformat()
    }
    
    # 保存到用户数据
    USER_DATA_STORE[get_or_create_user_id()]['co_create_history'].append(co_create_data)
    
    # 保存到数据库
    new_post = Post(
        user_id=get_or_create_user_id(),
        content_type=co_create_data['content_type'],
        content_data=str(co_create_data),
        location_name='共创生成',
        likes_count=0,
        comments_count=0
    )
    db.session.add(new_post)
    db.session.commit()
    
    return jsonify({
        'success': True,
        'message': '发布成功',
        'data': co_create_data
    })
```

```javascript
function displayGeneratedContent(data) {
    const resultContent = document.getElementById('generatedContent');
    
    if (data.title) {
        resultContent.innerHTML = `<strong>${data.title}</strong>\n\n${data.content}`;
    } else if (data.panels) {
        // 漫画脚本
        let text = '';
        data.panels.forEach((panel, index) => {
            text += `第${index + 1}格：${panel.scene}\n`;
            if (panel.dialogue) {
                text += `  对话：${panel.dialogue}\n`;
            }
            text += '\n';
        });
        resultContent.textContent = text;
    }
}
```
