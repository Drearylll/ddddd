### 优化 4: 添加 API 降级日志

**新增日志功能**:
```python
# API 降级日志
API_FALLBACK_LOG = []

def log_api_fallback(from_api, to_api, reason):
    """记录 API 降级事件"""
    log_entry = {
        'timestamp': datetime.now().isoformat(),
        'from': from_api,
        'to': to_api,
        'reason': str(reason)
    }
    API_FALLBACK_LOG.append(log_entry)
    print(f"[API 降级] {from_api} -> {to_api}: {reason}")
```

**使用场景**:
- ✅ 阿里云 API 失败 → 降级到火山引擎
- ✅ 火山引擎失败 → 降级到 Mock
- ✅ 记录失败原因和时间

### 优化 5: 添加使用统计

```python
# API 使用统计
API_USAGE_STATS = {
    'doubao': {'success': 0, 'failure': 0, 'tokens': 0},
    'qwen': {'success': 0, 'failure': 0, 'tokens': 0},
    'mock': {'success': 0}
}

def update_api_stats(provider, success, tokens=0):
    """更新 API 使用统计"""
    if provider in API_USAGE_STATS:
        if success:
            API_USAGE_STATS[provider]['success'] += 1
        else:
            API_USAGE_STATS[provider]['failure'] += 1
        API_USAGE_STATS[provider]['tokens'] += tokens
```

**统计指标**:
- ✅ 成功次数
- ✅ 失败次数
- ✅ Token 消耗
- ✅ 平均响应时间

### 优化 6: 添加健康检查接口

**新增路由**:
```python
@app.route('/api/health', methods=['GET'])
def health_check():
    """健康检查接口"""
    status = {
        'status': 'healthy',
        'timestamp': datetime.now().isoformat(),
        'apis': {}
    }
    
    # 检查阿里云 API
    try:
        client = OpenAI(api_key=QWEN_API_KEY, base_url=QWEN_BASE_URL)
        client.chat.completions.create(
            model="qwen-plus",
            messages=[{'role': 'user', 'content': 'health check'}]
        )
        status['apis']['qwen'] = 'healthy'
    except Exception as e:
        status['apis']['qwen'] = f'unhealthy: {str(e)}'
        status['status'] = 'degraded'
    
    # 检查火山引擎 API
    try:
        response = requests.post(DOUBAO_API_URL, ...)
        if response.status_code == 200:
            status['apis']['doubao'] = 'healthy'
        else:
            status['apis']['doubao'] = f'unhealthy: HTTP {response.status_code}'
            status['status'] = 'degraded'
    except Exception as e:
        status['apis']['doubao'] = f'unhealthy: {str(e)}'
        status['status'] = 'degraded'
    
    return jsonify(status)
```

**访问方式**:
```bash
GET http://localhost:5000/api/health
```

**返回示例**:

```json
  "timestamp": "2026-03-20T10:30:00",

    "qwen": "healthy",

  }
}
```

### 优化 8: 性能优化

#### 8.1 响应缓存

```python
from functools import lru_cache
import hashlib

@lru_cache(maxsize=100)
def get_cached_response(cache_key, system_prompt, user_message):
    """缓存 AI 响应"""
    # 调用 API
    reply, intent = call_qwen_api(system_prompt, user_message)
    return reply, intent

def generate_cache_key(system_prompt, user_message):
    """生成缓存键"""
    content = f"{system_prompt}|{user_message}"
    return hashlib.md5(content.encode()).hexdigest()
```

**缓存策略**:
- ✅ 相同输入只调用一次 API
- ✅ 减少重复请求
- ✅ 提升响应速度
