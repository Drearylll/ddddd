## 🔧 如何修改文件

### 修改 app.py

**步骤**:
1. 用 VS Code 或任何文本编辑器打开
2. 找到对应行号（参考本文档）
3. 修改代码
4. 保存并重启 Flask 程序

**注意**:
- 修改后必须重启程序才能生效
- 确保缩进正确（Python 对缩进敏感）
- 修改路由需要刷新浏览器缓存

### 修改 HTML 模板

**步骤**:
1. 用文本编辑器打开对应的.html 文件
2. 修改 HTML/CSS/JavaScript
3. 保存
4. 刷新浏览器即可看到效果（无需重启）

**注意**:
- CSS 在 `<style>` 标签内
- JS 在 `<script>` 标签内
- 可以使用浏览器开发者工具调试

## 📞 文件相关的问题排查

### 问题 1: 找不到模板文件
**错误**: `TemplateNotFound: onboarding.html`  
**解决**: 
1. 确认文件在 `templates/` 文件夹内
2. 检查文件名拼写是否正确
3. 重启 Flask 程序

### 问题 2: 修改后不生效
**原因**: 浏览器缓存  
**解决**: 
1. 强制刷新（Ctrl + Shift + R）
2. 清除浏览器缓存
3. 使用无痕模式

### 问题 3: 样式错乱
**原因**: CSS 文件未加载  
**解决**:
1. 检查 HTML 中的 `<link>` 标签
2. 确认 CSS 路径正确
3. 查看浏览器控制台是否有 404 错误
