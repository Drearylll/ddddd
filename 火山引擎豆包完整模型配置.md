## 💡 使用建议

### 文案生成任务

**推荐配置**：
```python
from services.doubao_ai import DoubaoAI

# 使用 Pro 版本获得最佳质量
ai = DoubaoAI()
result = ai.generate_text(
    prompt="写一段朋友圈文案",
    max_tokens=200,
    temperature=0.8
)
```

**快速响应场景**：
```python
# 使用 Lite 版本降低成本
ai_lite = DoubaoAI()
ai_lite.config = get_volcengine_config("text_lite")
```

### 图像理解任务

**推荐配置**：
```python
from services.doubao_ai import DoubaoAI

# 使用最强的视觉理解模型
ai_vision = DoubaoAI()
ai_vision.config = get_volcengine_config("vision")

result = ai_vision.analyze_image(
    image_url="https://example.com/image.jpg",
    question="请分析这张图片"
)
```

### 图像生成任务

**推荐配置**：
```python
from services.doubao_ai import DoubaoAI

ai_image = DoubaoAI()
ai_image.config = get_volcengine_config("image_generation")

result = ai_image.generate_image(
    prompt="上海外滩黄昏时分的美景",
    size="1080x1920"
)
```
