## ⚠️ 常见问题

### Q1: Invalid API-key provided

**原因**：
- API Key 格式错误
- 缺少 `sk-` 前缀
- Key 已过期或被禁用

**解决方法**：
1. 重新创建 API Key
2. 确保复制完整的 Key（包含 `sk-`）
3. 更新配置文件

### Q2: Service not activated

**原因**：
- 对应服务未开通
- 权限不足

**解决方法**：
1. 登录阿里云百炼控制台
2. 找到对应服务
3. 点击"开通服务"

### Q3: Insufficient balance

**原因**：
- 账户余额不足
- 免费额度已用完

**解决方法**：
1. 充值账户
2. 或等待下个月免费额度刷新

### Q4: Network timeout

**原因**：
- 网络连接问题
- API 服务器繁忙

**解决方法**：
1. 检查网络连接
2. 稍后重试
3. 如持续失败，联系阿里云客服
