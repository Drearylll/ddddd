### 1. 内容缓存机制
**实现位置：** `app.py` - `get_or_create_user_id()` 函数

**关键改动：**
- 在用户数据结构中增加 `'posts': []` 用于持久化存储已浏览内容

```python
USER_DATA_STORE[user_id] = {
```

### 2. 渐进生成机制
**实现位置：** `app.py` - `/` 路由（feed 函数）

**核心逻辑：**
1. **读取已有内容**：从用户数据中读取 `existing_posts`
2. **内容上限管理**：保留最近 30 条，超出删除最旧的
3. **渐进生成**：每次进入时新增 2-3 条新内容
4. **时间递进控制**：新内容时间必须晚于已有内容

**关键代码片段：**
```python
# 1. 读取用户已有的内容列表（持久化）
existing_posts = user_data.get('posts', [])

# 2. 内容上限管理：保留最近 30 条
MAX_POSTS = 30
if len(existing_posts) > MAX_POSTS:
    existing_posts = existing_posts[-MAX_POSTS:]

# 3. 渐进生成：每次进入时新增 2-3 条新内容
NUM_NEW_POSTS = random.randint(2, 3)

# 4. 时间递进控制
last_post_timestamp = user_data.get('last_post_timestamp')
if last_post_timestamp:
    # 新内容时间 = 最后一条时间 + 递增的分钟数
    time_offset = timedelta(minutes=random.randint(5 + i*10, 30 + i*10))
    post_timestamp = last_post_timestamp + time_offset
```

### 3. API 渐进生成支持
**实现位置：** `app.py` - `/api/next_post` 路由

**核心逻辑：**
1. 获取已有内容列表
2. 设置新内容时间戳（晚于已有内容）
3. 保存到用户数据
4. 执行内容上限管理

**关键代码片段：**
```python
# 获取已有内容列表
existing_posts = user_data.get('posts', [])

# 获取最后一条内容的时间戳（用于时间递进）
last_post_timestamp = user_data.get('last_post_timestamp')

# 设置时间戳：新内容时间必须晚于已有内容
if last_post_timestamp:
    new_post_timestamp = last_post_timestamp + timedelta(minutes=random.randint(5, 15))
else:
    new_post_timestamp = current_time

new_post['timestamp'] = new_post_timestamp

# 保存到已有内容列表（插入到最前面，因为最新）
existing_posts.insert(0, new_post)

# 内容上限管理：保留最近 30 条
MAX_POSTS = 30
if len(existing_posts) > MAX_POSTS:
    existing_posts = existing_posts[:MAX_POSTS]

save_user_data('posts', existing_posts)
save_user_data('last_post_timestamp', new_post_timestamp)
```

## 📊 数据流程

### 用户首次访问
```
创建用户 ID → 初始化 posts=[] → 生成 2-3 条初始内容 → 保存到 posts
```

### 用户再次访问
```
读取 posts → 保留最近 30 条 → 追加 2-3 条新内容 → 时间递进 → 保存更新
```

### 点击"继续"按钮
```
调用 /api/next_post → 生成 1 条新内容 → 设置递进时间 → 插入到 posts 顶部 → 保存
```

---

## ⏰ 时间逻辑

### 时间递进规则
1. **首次访问**：使用当前时间或过去几分钟
2. **再次访问**：新内容时间 = 最后一条时间 + 5~30 分钟
3. **点击继续**：新内容时间 = 最后一条时间 + 5~15 分钟

### 时间标签生成
```python
minutes_diff = int((post_timestamp - current_time).total_seconds() / 60)
if abs(minutes_diff) < 5:
    time_label = '刚刚'
elif abs(minutes_diff) < 60:
    time_label = f'{abs(minutes_diff)}分钟前'
else:
    hours_diff = int(abs(minutes_diff) / 60)
    time_label = f'{hours_diff}小时前'
```

## 📌 注意事项

1. **内存限制**：当前使用内存存储，重启后数据丢失
2. **多用户隔离**：每个用户独立数据，互不影响
3. **时间一致性**：确保所有时间戳都是 naive（无时区）
4. **内容排序**：始终按时间倒序排列（最新在上）
