# 阿里云百炼服务激活指南

## ⚠️ 当前问题

**错误代码**: `Arrearage`  
**错误信息**: Access denied, please make sure your account is in good standing  
**原因**: 账户未激活或欠费

---

## 🎯 激活步骤（必须完成）

### 步骤 1: 访问百炼控制台

🔗 地址：https://dashscope.console.aliyun.com/

**操作**:
1. 使用阿里云账号登录
2. 首次访问会提示开通服务

---

### 步骤 2: 开通百炼服务

**操作**:
1. 点击"开通服务"按钮
2. 阅读并同意服务协议
3. 确认开通

**注意**: 
- 即使有免费额度，也需要先开通服务
- 开通服务是免费的

---

### 步骤 3: 开通通义千问模型

**操作**:
1. 在控制台左侧菜单选择"模型广场"
2. 找到"通义千问"系列模型
3. 点击"开通模型"
4. 选择要开通的模型（建议开通多个）：
   - `qwen-turbo` - 基础版（便宜）
   - `qwen-plus` - 增强版（推荐）
   - `qwen-max` - 旗舰版

**推荐**: 至少开通 `qwen-turbo` 和 `qwen-plus`

---

### 步骤 4: 确认免费额度

**操作**:
1. 在控制台选择"资源管理" → "资源包"
2. 查看可用的免费额度
3. 确认额度详情

**新用户免费额度**（参考）:
- qwen-turbo: 一定额度的免费 token
- qwen-plus: 可能有体验额度
- 有效期：通常为开通后 30-180 天

---

### 步骤 5: 实名认证（如需要）

**某些情况下需要实名认证**:
1. 进入账号管理
2. 完成实名认证
3. 上传身份证或绑定支付宝

---

### 步骤 6: 验证 API Key

**操作**:
1. 在控制台选择"API-KEY 管理"
2. 确认您的 API Key 状态为"正常"
3. 如果状态异常，重新生成一个新的

**重新生成 API Key**:
1. 点击"创建新的 API-KEY"
2. 复制新的 Key
3. 更新到代码中

## 🔧 常见问题

### Q1: 显示"Arrearage"错误
**A**: 账户未激活或欠费
- 检查是否开通了百炼服务
- 检查是否开通了通义千问模型
- 查看账户余额

### Q2: 有免费额度但还是报错
**A**: 免费额度需要先激活服务
- 按照上述步骤开通服务
- 确认模型已开通
- 检查额度有效期

### Q3: API Key 无效
**A**: Key 可能过期或错误
- 在控制台重新生成 Key
- 更新到代码中
- 重启 Flask 服务

### Q4: 额度用完了怎么办
**A**: 
- 新用户通常有多次体验机会
- 可以注册新账号获取免费额度
- 或充值少量金额（很便宜）
