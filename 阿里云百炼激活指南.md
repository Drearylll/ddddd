## 🔍 常见问题

### 问题 1: 找不到"免费试用"

**解决方案**:
1. 直接访问：https://usercenter2.aliyun.com/free-trial
2. 或者在费用中心搜索"百炼"或"通义千问"
3. 如果没有免费额度，可能需要充值（最少 10 元）

### 问题 2: API Key 显示"已禁用"

**解决方案**:
1. 在 API Key 管理页面点击"启用"
2. 或者删除后重新创建一个新的 Key

### 问题 3: 提示"余额不足"

**解决方案**:
1. 进入费用中心 → 账户总览
2. 查看可用余额
3. 如果欠费，需要充值
4. 新用户通常有免费额度，检查是否领取

### 问题 4: 提示"未开通服务"

**解决方案**:
1. 返回百炼控制台首页
2. 点击"开通服务"
3. 同意协议并完成实名认证
