## 📋 部署步骤（以 Vercel 为例）

### 步骤 1：上传到 GitHub

```bash
# 初始化 git（如果还没有）
git init
git add .
git commit -m "Initial commit"

# 创建 GitHub 仓库并推送
git remote add origin https://github.com/你的用户名/goin.git
git push -u origin main
```

### 步骤 2：在 Vercel 部署

1. 访问 https://vercel.com
2. 点击 "New Project"
3. 选择 "Import Git Repository"
4. 选择你的 goin 仓库
5. 点击 "Deploy"

### 步骤 3：获取公网链接

部署完成后获得：
```
https://goin-xxx.vercel.app
```

这个链接可以直接分享给任何人体验！
