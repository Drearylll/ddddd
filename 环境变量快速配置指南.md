# 🚀 Vercel 环境变量 - 3 分钟快速配置指南

## ⚡ 最快速配置（仅需 3 个变量）

如果你只想快速测试，**只需配置 3 个环境变量**：

### 第 1 步：复制以下配置

```bash
FLASK_ENV=production
SECRET_KEY=goin_immersive_mvp_2026_secure_random_key_2026
GAODE_KEY=2274b3d46339f95092d68b83150ead7f
```

### 第 2 步：在 Vercel 控制台添加

1. 访问：https://vercel.com/dashboard
2. 找到 "goin" 项目 → Settings → Environment Variables
3. 点击 "Add New"
4. 逐个添加以上 3 个变量
5. 选择 "Production" 环境 ✅
6. 保存

### 第 3 步：重新部署

1. Deployments → 最新部署
2. 点击 "Redeploy"
3. 等待 2-5 分钟

**完成！** 🎉

## 🎯 使用 Vercel CLI 快速导入

### 安装 Vercel CLI

```bash
npm install -g vercel
```

### 登录

```bash
vercel login
```

### 创建 .env.local 文件

在项目根目录创建 `.env.local` 文件，粘贴上面的完整配置。

### 导入到 Vercel

```bash
vercel env pull .env.local
```

## 🆘 遇到问题？

### 环境变量未生效

1. 确认已添加到 "Production" 环境
2. 重新部署：Deployments → Redeploy
3. 清除缓存：勾选 "Clear Cache and Redeploy"

### API Key 无效

1. 检查 API Key 是否正确
2. 确认 API Key 已激活
3. 查看 API 控制台是否有余额

### 更多问题

查看 `Vercel 环境变量配置完整清单.md` 中的故障排查章节。
