## 🎨 五、视觉验收标准

### 5.1 色彩检查

```css
/* ✅ 正确使用 */
background: var(--bg-base);
color: var(--text-primary);
border: 1px solid var(--stroke-light);

/* ❌ 错误示范 */
background: #F7F7F7;  /* 应该用变量 */
color: #1A1A1A;       /* 应该用变量 */
```

### 5.2 间距检查

```css
/* ✅ 正确使用 */
padding: var(--space-4);
margin-bottom: var(--space-3);
gap: var(--space-2);

/* ❌ 错误示范 */
padding: 16px;  /* 应该用 --space-4 */
margin: 15px;   /* 不是 8 的倍数 */
```

### 5.3 动画检查

```css
/* ✅ 正确使用 */
transition: all 0.3s cubic-bezier(0.4, 0.0, 0.2, 1);
animation: fadeIn 0.3s ease-decelerate;

/* ❌ 错误示范 */
transition: all 0.3s ease;  /* 不够精确 */
animation: fade 300ms;      /* 单位不统一 */
```

## ⚡ 六、性能优化要点

### 6.1 CSS 优化

```css
/* GPU 加速 */
.optimized {
    transform: translateZ(0);
    will-change: transform;
    backface-visibility: hidden;
}

/* 避免重绘 */
.no-reflow {
    contain: layout style paint;
}
```

### 6.2 图片优化

```html
<!-- 懒加载 -->
<img loading="lazy" src="placeholder.jpg" data-src="real.jpg">

<!-- 响应式图片 -->
<picture>
    <source media="(min-width: 768px)" srcset="large.jpg">
    <img src="small.jpg" alt="Content">
</picture>
```

### 6.3 代码分割

```javascript
// 路由懒加载
const Feed = () => import('./pages/Feed.vue');
const Profile = () => import('./pages/Profile.vue');
```

## 💡 十、常见问题

### Q1: 如何确保风格统一？

**A**: 严格遵守设计系统中的变量使用，不要自定义颜色或间距。

### Q2: 动画卡顿怎么办？

**A**: 
1. 使用 GPU 加速（translateZ）
2. 减少同时进行的动画数量
3. 优化动画时长和缓动函数

### Q3: 如何适配不同屏幕？

**A**: 
1. 使用响应式单位（rem、vw、vh）
2. 媒体查询断点设置
3. iPhone 安全区域适配

### Q4: 暗黑模式如何实现？

**A**: CSS 文件中已预留，可在 `@media (prefers-color-scheme: dark)` 中定义暗黑主题变量。
