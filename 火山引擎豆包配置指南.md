在控制台中，找到"模型广场"或"我的模型"

**查找以下类型的模型**：

#### 图像生成模型
可能的名称：
- `doubao-text2image` - 豆包文生图
- `seed-xxx` - 可灵模型

### 步骤 5：开通服务

1. 点击需要的模型（如 doubao-pro-4k）
2. 点击"开通服务"
3. 阅读并同意服务协议
4. 确认开通

**建议开通的模型**：
- ✅ doubao-pro-4k（文本生成）
- ✅ doubao-vision-pro-32k（视觉理解）
- ✅ doubao-text2image（图像生成）

### 步骤 6：更新配置文件

根据控制台显示的模型名称，更新配置文件：

```python

    "image_generation": {
        "model": "这里填写图像生成模型的实际名称",
        "api_key": DOUBAO_API_KEY,
        "base_url": DOUBAO_BASE_URL

```

## ⚠️ 常见问题

### Q1: "The model or endpoint xxx does not exist"

**原因**：
- 模型名称填写错误
- 该模型对你不可用
- 服务未开通

**解决方法**：
1. 登录火山引擎控制台
2. 在"模型广场"查看可用的模型
3. 复制准确的模型 ID（区分大小写）
4. 更新配置文件

### Q2: "You do not have access to it"

**原因**：
- API Key 没有该模型的访问权限
- 服务未开通
- 账户欠费或余额不足

**解决方法**：
1. 检查 API Key 是否正确
2. 在控制台开通对应服务
3. 检查账户余额

### Q3: 找不到对应的模型

**可能情况**：
- 该模型已下线
- 模型名称已变更
- 你的账号没有该模型的访问权限

**解决方法**：
1. 查看火山引擎官方文档
2. 联系火山引擎技术支持
3. 或者暂时使用阿里云百炼作为替代
