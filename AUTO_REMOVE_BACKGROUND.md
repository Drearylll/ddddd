# 🤖 AI 自动抠图工具

## 📸 功能说明

这是一个**全自动抠图工具**，使用 AI 模型自动去除人物照片背景，生成透明 PNG 图片。

## 🚀 使用步骤

### 步骤 1：准备 JPG 文件

将您的人物照片（z1.jpg 到 z9.jpg）放入目录：
```
static/images/characters/
```

### 步骤 2：运行自动抠图脚本

在项目根目录运行：
```bash
python auto_remove_background.py
```

### 步骤 3：等待处理完成

脚本会自动：
1. 扫描所有 JPG 文件
2. 使用 AI 去除背景
3. 生成透明 PNG 文件
4. 保存在同一目录

### 步骤 4：刷新浏览器

清除缓存并刷新：
```
Ctrl + Shift + R
```

## 🎨 处理效果

**处理前（JPG）**：
```
┌─────────────┐
│ 人物 + 背景 │
└─────────────┘
```

**处理后（PNG）**：
```
     ┌─────────────┐
     │   人物      │ ← 透明背景
     └─────────────┘
```

---

## 💡 技术原理

**使用 rembg 库**：
- 基于 U-2-Net 深度学习模型
- 自动识别人物主体
- 精确去除背景
- 保留头发等细节

**处理速度**：
- 单张：约 1-3 秒
- 9 张：约 10-30 秒

**输出文件**：

## 💻 高级用法

### 单次处理一张
```bash
python -c "from rembg import remove; from PIL import Image; img = Image.open('static/images/characters/z1.jpg'); output = remove(img); output.save('static/images/characters/z1.png')"
```

### 批量处理所有 JPG
```bash
python auto_remove_background.py
```

## ⚠️ 注意事项

1. **首次运行会下载模型**（约 100MB）
2. **需要网络连接**（仅首次）
3. **处理速度取决于 CPU**
4. **建议图片尺寸**：400x600 到 800x1200

---

## 🎨 优化建议

**图片要求**：
- 人物清晰
- 背景对比度明显
- 避免复杂背景
- 人物与背景分离明显

**效果最佳**：
- 背影照片
- 侧脸照片
- 远景照片
- 有遮挡的照片
