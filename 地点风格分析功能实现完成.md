### 函数签名

```python
def analyze_location_style(
```

```json
}
```

### 使用示例

```python
from services.location_analyzer import analyze_location_style

# 分析咖啡厅
result = analyze_location_style(
    image_url="https://images.pexels.com/photos/302899/pexels-photo-302899.jpeg",

)

print(result['color_tone'])  # 暖色
```
