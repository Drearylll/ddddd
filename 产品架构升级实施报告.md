### 1. UserService - 用户服务 ⭐⭐⭐⭐⭐

#### 核心方法

```python
UserService.create_user_id()           # 创建唯一用户 ID
UserService.generate_avatar(photo, gender)  # AI 生成二维手绘头像
UserService.validate_user_data(data)   # 验证用户数据
UserService.save_avatar_to_file(avatar, user_id)  # 保存头像
```

#### 头像生成流程

```python
用户上传照片
    ↓
PIL 打开图片 → 512x512 调整
    ↓
应用卡通风格滤镜
    ├─ 边缘增强
    ├─ 色彩量化（64 色）
    ├─ 饱和度增强 +50%
    └─ 亮度增强 +10%
    ↓
根据性别调整色调
    ├─ 女性：暖色调（红 + 黄）
    └─ 男性：冷色调（蓝）
    ↓
手绘效果处理
    ├─ 轻微模糊
    └─ 锐化边缘
    ↓
保存为 base64 或文件
```

#### 技术亮点

- ✅ **唯一性保证**：UUID 生成唯一头像 ID
- ✅ **卡通风格**：多步骤图像处理
- ✅ **性别区分**：不同色调处理
- ✅ **高质量输出**：512x512 分辨率
- ✅ **灵活存储**：支持 base64 和文件两种格式

#### 1. 用户相关路由

```python
@app.route('/register')
def register():
    """注册页面"""
    return render_template('register.html')

@app.route('/api/register', methods=['POST'])
def api_register():
    """注册 API"""
    data = request.get_json()
    user = UserService.create_user(data)
    return jsonify({'success': True, 'user': user})

@app.route('/upload_avatar', methods=['POST'])
def upload_avatar():
    """上传头像 API"""
    photo = request.files['photo']
    gender = request.form['gender']
    avatar = UserService.generate_avatar(photo, gender)
    return jsonify({'success': True, 'avatar': avatar})
```

from concurrent.futures import ThreadPoolExecutor

executor = ThreadPoolExecutor(max_workers=4)

def generate_avatar_async(photo, gender):
    """异步生成头像"""
    future = executor.submit(UserService.generate_avatar, photo, gender)
    return future.result()
