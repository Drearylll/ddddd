#### 新增路由 ✅
```python
@app.route('/')                              # 首页（模式选择）
@app.route('/feed')                          # Web 体验（原有内容流）
@app.route('/onboarding')                    # 欢迎引导

@app.route('/api/generate_moment')                         # 生成朋友圈
```
