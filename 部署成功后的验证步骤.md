## 🎯 问题排查

### 如果部署失败

#### 错误 1：Could not import app.py

**原因**：依赖未正确安装

**解决方案**：
```bash
# 清除 Vercel 缓存
1. Settings → Git → Ignored Build Step → 设置为空
2. Deployments → 最新部署 → Redeploy
3. 勾选 "Clear Cache and Redeploy"
```

#### 错误 2：ModuleNotFoundError: No module named 'xxx'

**原因**：缺少依赖

**解决方案**：
1. 检查 `requirements.txt` 是否包含该模块
2. 确认版本号正确
3. 重新部署

#### 错误 3：Database connection failed

**原因**：数据库配置问题

**解决方案**：
```python
# config/db_config.py 已自动处理
# Vercel 环境会使用内存数据库
DATABASE_PATH = ':memory:'
```
