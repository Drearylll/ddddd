---

### 可能原因 2：数据库初始化问题

**错误日志**:
```
sqlite3.OperationalError: unable to open database file
```

**解决方案**: 已在 app.py 中实现 Vercel 环境适配

```python
# Vercel 环境强制使用内存数据库
if VERCEL_ENV:
    app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///:memory:'
```

**验证**: 访问 `/debug-status` 查看数据库配置

---

### 可能原因 3：环境变量缺失

**错误日志**:
```
KeyError: 'SECRET_KEY'
```

**解决方案**: ✅ 已在 app.py 中实现防御性导入

```python
SECRET_KEY = os.getenv('SECRET_KEY', 'fallback_key')
```

---

### 可能原因 4：服务导入失败

**错误日志**:
```
ModuleNotFoundError: No module named 'services.xxx'
```

**解决方案**: ✅ 已实现延迟导入机制

```python
# services/__init__.py 使用 __getattr__
def __getattr__(name):
    # 延迟导入，避免部署失败
```
