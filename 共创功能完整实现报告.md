## ✅ 已实现的完整流程

### 🎯 核心功能

**用户流程**：
```
点击共创 → 选择模式 → 输入要求 → AI 生成 → 查看结果 → 重新生成/发布到看看 → 看看同步显示
```

---

## 📊 三种步骤详解

### 步骤 1️⃣: 输入要求

**界面元素**：
- 原内容展示（标题 + 内容）
- 模式徽章（接力/夺擂）
- 输入框（描述创作要求）
- 按钮：取消 / 开始生成

**接力模式提示**：
```
✏️ 请续写下一段/下一格（描述您的想法）
例如：主角走进了一家神秘的书店，发现了一本奇怪的书...
```

**夺擂模式提示**：
```
💡 请描述您想要的新内容（AI 将根据要求生成）
例如：一个关于重逢的故事，发生在黄昏的地铁站...
```

---

### 步骤 2️⃣: AI 生成

**界面显示**：
- 🤖 AI 正在创作
- 加载动画
- 调用通义千问 API

**API 调用**：
```javascript
POST /api/co_create_generate
{
    "prompt": "用户输入的要求",
    "type": "novel" 或 "comic",
    "original_content": "原内容"
}
```

**生成逻辑**：
- 小说类型：调用 `generate_novel()` 方法
- 漫画类型：调用 `generate_comic_script()` 方法
- 使用用户输入作为提示词

---

### 步骤 3️⃣: 发布作品

**界面显示**：
- ✨ 生成结果（标题 + 内容）
- 按钮：重新生成 / 发布到看看

**重新生成**：
- 清空输入框
- 返回步骤 1
- 可重新输入要求

**发布到看看**：
```javascript
POST /api/publish_co_create
{
    "content_type": "novel/comic",
    "title": "作品标题",
    "content": "作品内容",
    "mode": "relay/challenge",
    "original_content": "原内容",
    "user_contribution": "用户输入的要求"
}
```

**发布后动作**：
1. 保存到数据库（Post 表）
2. 保存到用户共创历史
3. 设置 localStorage 标记
4. 跳转到看看页面
5. 自动显示新内容

async function generateContent() {
    const prompt = document.getElementById('userPrompt').value.trim();

                prompt: prompt,
                type: contentType,
                original_content: originalContent
            })
        });
        
        const data = await response.json();
        
        if (data.success) {
            generatedData = data.data;
            displayGeneratedContent(data.data);
            
            // 切换到步骤 3

### 后端：生成 API

```python
@app.route('/api/co_create_generate', methods=['POST'])
def api_co_create_generate():
    """共创内容生成 API"""
    try:
        data = request.get_json() or {}
        prompt = data.get('prompt', '')  # 用户输入的要求
        content_type = data.get('type', 'novel')  # novel 或 comic
        original_content = data.get('original_content', '')  # 原内容
        
        from services.ai_content import get_ai_service
        ai_service = get_ai_service()
        
        # 根据用户要求生成内容
        if content_type == 'novel':
            result = ai_service.generate_novel(
                location='附近',
                mood='日常',
                user_profile=prompt  # 使用用户输入作为提示
            )
        elif content_type == 'comic':
            result = ai_service.generate_comic_script(
                location='附近',
                theme=prompt  # 使用用户输入作为主题
            )
        else:
            result = {'error': 'Unknown content type'}
        
        return jsonify({
            'success': True,
            'type': content_type,
            'data': result
        })
        
    except Exception as e:
        print(f"[ERROR] Failed to generate co-create content: {e}")
        return jsonify({'error': str(e)}), 500
```

### 后端：发布 API

```python
@app.route('/api/publish_co_create', methods=['POST'])
def api_publish_co_create():
    """发布共创内容到看看页面"""
    try:
        data = request.get_json() or {}
        user_data = get_user_data()
        
        # 获取共创内容
        co_create_data = {
            'content_type': data.get('content_type', 'novel'),
            'title': data.get('title', '共创作品'),
            'content': data.get('content', ''),
            'mode': data.get('mode', 'relay'),
            'original_content': data.get('original_content', ''),
            'user_contribution': data.get('user_contribution', ''),
            'created_at': datetime.now().isoformat()
        }
        
        # 添加到用户数据
        if 'co_create_history' not in USER_DATA_STORE.get(get_or_create_user_id(), {}):
            USER_DATA_STORE[get_or_create_user_id()]['co_create_history'] = []
        
        USER_DATA_STORE[get_or_create_user_id()]['co_create_history'].append(co_create_data)
        
        # 保存到数据库
        try:
            from services.database import Post
            new_post = Post(
                user_id=get_or_create_user_id(),
                content_type=co_create_data['content_type'],
                content_data=str(co_create_data),
                location_name='共创生成',
                likes_count=0,
                comments_count=0
            )
            db.session.add(new_post)
            db.session.commit()
        except Exception as e:
            print(f"⚠️ 数据库保存失败：{e}")
        
        return jsonify({
            'success': True,
            'message': '发布成功',
            'data': co_create_data
        })
        
    except Exception as e:
        print(f"[ERROR] Failed to publish co-create content: {e}")
        return jsonify({'error': str(e)}), 500
```
