## 💡 五、使用建议

### 5.1 最佳实践

#### ✅ 推荐做法
1. 使用 PNG 格式（支持透明）
2. 提供多分辨率版本（@1x, @2x, @3x）
3. 添加适当的阴影效果
4. 保持安全间距
5. 在浅色背景使用

#### ❌ 避免做法
1. 拉伸变形（保持比例）
2. 修改颜色（保持紫色渐变）
3. 添加过多效果（保持简洁）
4. 旋转使用（保持水平）
5. 拆分使用（保持完整）

---

### 5.2 响应式尺寸

```css
/* 手机 */
@media (max-width: 768px) {
    .logo-image {
        max-width: 240px;
    }
}

/* 平板 */
@media (min-width: 769px) and (max-width: 1024px) {
    .logo-image {
        max-width: 320px;
    }
}

/* 桌面 */
@media (min-width: 1025px) {
    .logo-image {
        max-width: 400px;
    }
}
```
