## 🎯 项目概况（快速版）

### 这是什么？
**Go In** - AI 社交 App，核心理念：
- 自动生成用户在真实地点的生活内容
- 展现"另一个可能的自己"
- 用户是观察者，不是操作者

### 技术栈
- **后端**: Flask 3.0.0 (Python 3.12)
- **前端**: 纯 HTML/CSS/JS (无框架)
- **部署**: Vercel Serverless
- **数据库**: SQLite + SQLAlchemy

### 核心文件
```
app.py                          # 主应用（~3600 行）
templates/profile_setup.html    # 头像设置页面
templates/app_main.html         # 主界面
services/                       # 业务逻辑层
vercel.json                     # Vercel 配置
```

## 🚀 快速开始示例

### 示例 1: 添加一个新页面

**需求**: 添加"设置"页面

**步骤**:

1. **创建 HTML 模板**
```html
<!-- templates/settings.html -->
<!DOCTYPE html>
<html>
<head>
    <title>设置 - Go In</title>
    <link rel="stylesheet" href="/static/css/goin_brand_v3.css">
</head>
<body>
    <div class="container">
        <h1>设置</h1>
        <!-- 设置内容 -->
    </div>
</body>
</html>
```

2. **添加路由**
```python
# app.py
@app.route('/settings')
def settings():
    return render_template('settings.html')
```

3. **添加访问入口**
```html
<!-- 在 app_main.html 中添加链接 -->
<a href="/settings" class="btn">设置</a>
```

### 示例 2: 修复一个 bug

**问题**: 按钮点击无效

**步骤**:

1. **找到相关代码**
```bash
# 搜索按钮 ID 或 class
grep -r "button-id" templates/
```

2. **查看事件绑定**
```javascript
// 检查 JavaScript
document.getElementById('button-id')
```

3. **修复代码**
```javascript
// 添加事件监听
document.getElementById('button-id')
    .addEventListener('click', handleClick);
```

4. **测试验证**
- 推送代码
- 访问页面测试
