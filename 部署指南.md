# Go In 公网部署指南

## 一、快速部署方案（推荐）

### 方案 1：Vercel 部署（最简单，免费）

#### 1. 准备文件

在项目根目录创建以下文件：

**vercel.json**
```json
{
  "version": 2,
  "builds": [
    {
      "src": "app.py",
      "use": "@vercel/python"
    }
  ],
  "routes": [
    {
      "src": "/(.*)",
      "dest": "app.py"
    }
  ],
  "env": {
    "FLASK_ENV": "production"
  }
}
```

**requirements.txt**
```
Flask==3.0.0
gunicorn==21.2.0
```

---

#### 2. 部署步骤

**步骤 1：上传代码到 GitHub**
```bash
# 初始化 git（如果还没有）
git init
git add .
git commit -m "Initial commit"

# 创建 GitHub 仓库并推送
git remote add origin https://github.com/你的用户名/goin.git
git push -u origin main
```

**步骤 2：在 Vercel 导入项目**
1. 访问 https://vercel.com
2. 点击 "New Project"
3. 选择 "Import Git Repository"
4. 选择你的 goin 仓库
5. 点击 "Deploy"

**步骤 3：获取公网链接**
- 部署完成后，Vercel 会生成一个链接：`https://goin-xxx.vercel.app`
- 这个链接可以直接分享给任何人

---

#### 3. 修改 app.py 支持 Vercel

在 `app.py` 最后添加以下代码：

```python
# Vercel 生产环境支持
if __name__ == '__main__':
    # 本地开发环境
    app.run(debug=True, host='0.0.0.0', port=5000)
```

Vercel 会自动使用 gunicorn 运行 Flask 应用，不需要修改。

### 方案 3：Render 部署（免费，稳定）

#### 1. 准备文件

**requirements.txt**（同上）
```
Flask==3.0.0
gunicorn==21.2.0
```

---

#### 2. 部署步骤

1. 访问 https://render.com
2. 注册账号
3. 点击 "New +" → "Web Service"
4. 连接 GitHub 仓库
5. 配置：
   - **Name:** goin
   - **Environment:** Python
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `gunicorn app:app`
6. 点击 "Create Web Service"

部署完成后获得链接：`https://goin.onrender.com`

### 方案 4：云服务器部署（阿里云/腾讯云）

#### 1. 购买服务器

- **阿里云：** 选择 "轻量应用服务器"（约 99 元/年）
- **腾讯云：** 选择 "轻量应用服务器"（约 88 元/年）
- **配置建议：** 1 核 2G，Ubuntu 20.04

---

#### 2. 服务器配置

**步骤 1：SSH 连接服务器**
```bash
ssh root@你的服务器 IP
```

**步骤 2：安装 Python 和依赖**
```bash
# 更新系统
apt update
apt upgrade -y

# 安装 Python 3 和 pip
apt install python3 python3-pip -y

# 安装 Nginx
apt install nginx -y
```

**步骤 3：上传代码**
```bash
# 方法 1：使用 git
cd /var/www
git clone https://github.com/你的用户名/goin.git
cd goin

# 方法 2：使用 scp（在本地执行）
scp -r ./ root@你的服务器 IP:/var/www/goin
```

**步骤 4：安装依赖**
```bash
cd /var/www/goin
pip3 install flask gunicorn
```

**步骤 5：配置 Gunicorn**
```bash
# 创建 systemd 服务文件
nano /etc/systemd/system/goin.service
```

内容：
```ini
[Unit]
Description=Goin Flask Application
After=network.target

[Service]
User=root
WorkingDirectory=/var/www/goin
ExecStart=/usr/bin/gunicorn -w 4 -b 0.0.0.0:8000 app:app
Restart=always

[Install]
WantedBy=multi-user.target
```

**步骤 6：启动服务**
```bash
systemctl daemon-reload
systemctl start goin
systemctl enable goin
```

**步骤 7：配置 Nginx**
```bash
nano /etc/nginx/sites-available/goin
```

内容：
```nginx
server {
    listen 80;
    server_name 你的域名或 IP;

    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

启用配置：
```bash
ln -s /etc/nginx/sites-available/goin /etc/nginx/sites-enabled/
nginx -t
systemctl restart nginx
```

**步骤 8：访问**
- 通过 `http://你的服务器 IP` 访问
